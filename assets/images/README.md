# Images Directory

Place your café images here.

## Recommended Images

### Hero Section Background
- **Filename:** `hero-bg.jpg`
- **Size:** 1920x1080px (Full HD)
- **Usage:** Background image for the hero section
- **Tips:** Use a warm, inviting photo of your café or food

### Menu Item Images (Optional)
- **Location:** `assets/images/menu/`
- **Naming:** Use lowercase with hyphens (e.g., `cheese-burger.jpg`)
- **Size:** 400x400px (square)
- **Format:** JPG or PNG

### About Section
- **Filename:** `cafe-interior.jpg` or `cafe-exterior.jpg`
- **Size:** 800x600px
- **Usage:** Show your café's ambiance

### Logo (Optional)
- **Filename:** `logo.png`
- **Size:** 200x200px
- **Format:** PNG with transparent background
- **Usage:** Can be added to navbar

## Image Optimization Tips

1. **Compress images** before uploading (use TinyPNG or similar)
2. **Use appropriate formats:**
   - JPG for photos
   - PNG for logos/graphics with transparency
   - WebP for modern browsers (optional)
3. **Keep file sizes under 500KB** for fast loading

## How to Add Images

1. Upload images to this directory
2. Reference them in your HTML/CSS:
   ```html
   <img src="assets/images/your-image.jpg" alt="Description">
   ```
   ```css
   background-image: url('../assets/images/your-image.jpg');
   ```
